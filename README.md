# WebStudio

Responsive landing page developed using HTML5 and CSS3.

## Features

- Semantic HTML markup with SVG icon sprite
- Flexbox layout with CSS custom properties
- Modal window with backdrop and contact form
- Custom checkbox, focus/hover transitions on all interactive elements
- Footer newsletter subscription form

## Technologies

- HTML5
- CSS3
- Google Fonts (Roboto, Raleway)
- Prettier
